variables_analisis = ['Fresh', 'Milk', 'Grocery', 'Frozen', 'Detergents_Paper', 'Delicassen']

# Realiza el test de ANOVA para cada variable
for variable in variables_analisis:
    muestras = [canal1[variable], canal2[variable]]
    
    # Realiza el test de ANOVA
    stat, p_valor = f_oneway(*muestras)
    
    alpha = 0.05
    print("Variable:", variable)
    print("Estadstico de prueba:", stat)
    print("Valor p:", p_valor)
    if p_valor > alpha:
        print("No hay diferencias significativas entre los canales para la variable", variable)

for variable_objetivo in variables_analisis:
    # Seleccionar la variable objetivo actual
    y = consumo[variable_objetivo]
    
    # Seleccionar las caractersticas (X) correspondientes
    X = consumo.drop(['Channel', 'Region'], axis=1)
    
    # Dividir los datos en conjuntos de entrenamiento y prueba
    X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=58)
    
    # Inicializar el modelo de regresin lineal
    model = LinearRegression()
    
    # Entrenar el modelo
    model.fit(X_train, y_train)
    
    # Hacer predicciones en el conjunto de prueba
    y_pred = model.predict(X_test)
    
    # Calcular el error cuadrtico medio (MSE)
    mse = mean_squared_error(y_test, y_pred)
    print("Error cuadrtico medio para", variable_objetivo, ":", mse)
    
    # Imprimir los coeficientes del modelo para ver su importancia
    print("Coeficientes del modelo para", variable_objetivo, ":", model.coef_)
    print()

Para realizar el anlisis predictivo, se escogi un modelo de regresin lineal, ya que es el ms comn y, al no tener datos temporales, es el ms acertado.

El modelo intenta explicar una variable en funcin de las dems variables presentes en el modelo. La idea es encontrar relaciones que nos permitan vincular la variable objetivo con las dems.

En este caso, no fue posible, ya que ninguno de los valores que arroja el modelo para cada una de las variables correspondientes result significativo, excepto para la variable en s misma, pero ese resultado no nos dice mucho. Esto se puede explicar porque la mayora de las variables tienen una alta desviacin estndar, lo que hace difcil llegar a conclusiones vlidas.

Al momento de realizar las regresiones, se segmentaron los datos por canal y regin para cada una de las variables en particular, pero en todos los casos los valores eran similares y no significativos, por lo que se dej el dataframe completo en el resultado final.

Conclusiones

Los datos proporcionados nos muestran los patrones de consumo anuales para ciertos productos en particular, del anlisis podemos concluir.

  1. Los productos ms consumidos son los productos frescos. Sera necesario realizar un anlisis ms profundo para decidir si los dems productos requieren una campaa publicitaria o para entender por qu los productos frescos estn tan bien posicionados.
  2. Los datos muestran una dispersin considerable, con desviaciones estndar ms altas que la media, lo que complica la validez de los modelos predictivos.
  3. Para ambos canales, los productos delicatessen tienen consumos similares en las tres regiones. Sera interesante analizar este segmento ms a fondo para encontrar la razn en particular.
  4. En el caso del canal Horeca, la matriz de correlacin muestra una fuerte relacin entre los lcteos y los productos delicatessen, as como tambin con los productos comestibles. Por lo tanto, organizar estos productos en los establecimientos de tal forma que los lcteos estn cerca de estos dos productos sera una buena estrategia para incentivar las ventas.
  5. Para el canal minorista, existe una estrecha relacin entre los productos de detergentes y papel y los comestibles. Sin embargo, colocarlos muy cerca no es necesario debido a la alta relacin. Sera ms efectivo realizar campaas de promocin entre lcteos y comestibles, ya que tendran un buen impacto en las ventas de ambos.
  6. Los datos no siguen una distribucin normal y algunos tests no son muy confiables.
